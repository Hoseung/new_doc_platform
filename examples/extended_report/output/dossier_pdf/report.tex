\subsection{8. Recommendations}\label{recommendations}

Based on the analysis, we recommend the following improvements:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \textbf{Sunglasses Handling}: Implement adaptive IR intensity for
  sunglasses detection
\item
  \textbf{Gaze Calibration}: Add per-driver gaze calibration for
  improved accuracy
\item
  \textbf{Latency Optimization}: Target 15ms reduction through model
  pruning
\item
  \textbf{Edge Case Coverage}: Expand training data for identified
  failure modes
\end{enumerate}

\subsection{Appendix A: Methodology}\label{appendix-a-methodology}

\subsubsection{A.1 Dataset Description}\label{a.1-dataset-description}

The evaluation dataset (ds\_dms\_eval\_2024\_v2) contains: - 150,000
annotated frames - 500 unique subjects - 12 lighting conditions - 8
vehicle types

\subsubsection{A.2 Evaluation Protocol}\label{a.2-evaluation-protocol}

All metrics were computed using: - 5-fold cross-validation - Stratified
sampling by subject - Held-out test set (20\% of data)
